## Install Dependencies

1. Run `npm install` in app folder and root directory
2. Download Ganache and add truffle-config.js to it
3. Install Truffle globally


## Start Development Server 
Run in root directoy:
```
Truffle compile
Truffle migrate
```

Run in app folder:

`npm run start`
